# Интенсив по Аналитике: Презентация

Сайт-презентация для 3-дневного интенсива по аналитике на датасете аптечной сети.

## Описание

Интенсив охватывает три ключевых инструмента аналитика:
- **День 1**: Таблицы (Excel/Google Sheets) — быстрый анализ
- **День 2**: SQL — масштабируемые отчёты
- **День 3**: Python — визуализация и storytelling

## Как использовать

1. Откройте `index.html` в браузере
2. Используйте навигацию для перемещения между слайдами

## Навигация

- **Кнопки**: "Назад" и "Далее" в правом нижнем углу
- **Клавиатура**: 
  - `→` или `Пробел` — следующий слайд
  - `←` — предыдущий слайд
  - `Home` — первый слайд
  - `End` — последний слайд
  - `Ctrl+F` — полноэкранный режим
- **Точки прогресса**: клик по точке переходит к соответствующему слайду

## Интерактивные элементы

- Кликабельные карточки с раскрывающимися деталями
- Копирование кода одним кликом
- Счётчик слайдов в левом нижнем углу

## Внешние инструменты

- [Google Sheets](https://sheets.google.com) — для практики День 1
- [CSVFiddle](https://csvfiddle.io) — для практики SQL День 2
- [Google Colab](https://colab.research.google.com) — для практики Python День 3
